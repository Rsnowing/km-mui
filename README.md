# km-mui
## npm 安装
```
npm i --save km-mui
```

## 项目启动
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```
